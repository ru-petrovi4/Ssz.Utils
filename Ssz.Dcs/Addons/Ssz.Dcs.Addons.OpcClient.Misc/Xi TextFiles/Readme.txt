The files in this directory are sample files that are used in the Xi Solution.

Xi paths.txt
This file contains paths to data objects used when the "Add From File" button is 
pushed on the DataLists tab of the TabbedServerDlg in the Xi Client GenericApp VS2008 
project. The file that contains these data object instance ids can have any name 
and can be located anywhere since the "Add From File" button allows browsing for the 
file.
